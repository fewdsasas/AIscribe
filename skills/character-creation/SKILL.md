# 角色塑造技能

## 功能
设计有深度、有弧光的立体角色，涵盖性格、背景、动机和关系网络。

## 核心模块
1. **性格设计** — MBTI、九型人格、核心特质
2. **角色弧光** — 正面弧、负面弧、静态弧
3. **背景故事** — 过去塑造现在
4. **动机与目标** — 欲望 vs 需求
5. **缺陷与恐惧** — 让角色真实可信
6. **关系网络** — 角色之间的连接
7. **对话风格** — 为每个角色设计独特的声音

## 角色类型
- 主角(Protagonist)、反派(Antagonist)、恋爱对象(Love Interest)
- 导师(Mentor)、伙伴(Sidekick)、对手(Foil)
- 配角(Supporting)、知己(Confidant)、反派(Villain)
