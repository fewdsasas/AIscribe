# 故事结构技能

## 功能
为小说创作提供专业的叙事结构设计，支持多种经典框架。

## 支持的框架
1. **三幕剧** — 最经典的结构，分为开端(Setup)、对抗(Confrontation)、解决(Resolution)
2. **英雄之旅** — 12个阶段的英雄成长弧线
3. **Save the Cat** — 15个节拍点，适合电影化叙事
4. **七点结构** — Hook(钩子) → Plot Turn 1 → Pinch 1 → Midpoint → Pinch 2 → Plot Turn 2 → Resolution
5. **雪花法** — 从一句话到完整大纲的渐进式扩展
6. **故事圈** — 基于角色弧线的循环结构
7. **故事网格** — 基于类型的惯例和必备场景
8. **戏剧理论** — 四幕结构，关注角色动机和冲突

## 使用方法
- 分析现有故事的结构
- 为新项目选择合适的框架
- 生成节拍表和时间线
- 检查结构的完整性
