#### 任务 3.2：修复测试代码中的 no-non-null-assertion（~139 处）
- **文件**: 20+ 个测试文件
- **方法**: 
  - `mockHandlers.get('ch')!` → `const h = mockHandlers.get('ch'); expect(h).toBeDefined(); if (!h) return;`

  - `config!.apiKey` → `expect(config).toBeDefined(); if (!config) return; config.apiKey`
